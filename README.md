oeed :muscle:
====

Open Engineer Energy Drink.

材料(分量: 1l) :honey_pot:
------

* 炭酸水 750ml
* フレーバーシロップ(メロン) 250 ml
* カフェイン(錠剤) 600mg
* コブラ粉末 350mg
* 牡蠣エキス  350mg
* 冬虫夏草粉末 350mg
* 田七人参粉末 350mg
* エゾウコギ粉末 350mg
* マカ粉末 350mg
* ガラナ粉末 350mg

作り方 :cookie:
------

1. カフェイン(錠剤)をすり鉢等ですり潰す
2. 容器に、フレーバーシロップを入れて、炭酸水で4倍に希釈する
3. 2に、粉末状になったカフェインと各種粉末を入れる
4. よくかき混ぜる

注意 :fire:
------

* よく冷やしてお召し上がり下さい :tropical_drink:
